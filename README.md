# User Management System (Java)

## Описание
Данный проект представляет собой простую систему управления пользователями
для небольшой компании. Реализация выполнена с применением принципов
YAGNI, KISS и DRY.

Система позволяет:
- добавлять пользователей
- удалять пользователей
- обновлять данные пользователя

Каждый пользователь имеет:
- имя
- email
- роль (Admin / User)

---

## Применённые принципы проектирования

### YAGNI (You Aren't Gonna Need It)
- Реализована только необходимая функциональность
- Нет поиска по множеству критериев
- Нет интерфейсов, абстрактных классов и расширенных ролей
- Нет работы с БД, файлами или сетью

### KISS (Keep It Simple, Stupid)
- Простая линейная логика
- Минимальное количество условий
- Используется List<User>
- Код легко читается и понимается

### DRY (Don't Repeat Yourself)
- Логика поиска пользователя вынесена в один метод `findByEmail`
- Обновление данных пользователя выполняется через один метод `update`
- Нет дублирования кода при добавлении, удалении и обновлении
